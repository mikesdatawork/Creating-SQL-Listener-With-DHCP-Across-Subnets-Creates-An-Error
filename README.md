## About-Process

<p>The short answer is 'No'. You will run into this error:
Msg 19458, Level 16, State 1, Line 1
The WSFC nodes that host the primary and secondary replicas belong to different subnets. 
DHCP across multiple subnets is not supported for availability replicas. 
Use the static IP option to configure the availability group listener.</p>
